# 🎨 Create Social Media Sharing Image

## Required: og-image.png for Social Media Sharing

To complete the SEO setup, you need to create a social media optimized image:

### Image Specifications
- **File name**: `og-image.png`
- **Dimensions**: 1200px × 630px (optimal for all platforms)
- **Location**: Save in `/public/og-image.png`
- **File size**: Under 5MB (preferably under 1MB)

### Design Guidelines
1. **Include Pecha.tools logo/branding**
2. **Buddhist-themed background** (subtle patterns, mandala, or gradients)
3. **Text overlay**: "AI Tools for Buddhist Text Collection & Translation"
4. **Color scheme**: Use the app's indigo theme (#4f46e5)
5. **Font**: Clean, readable typography (similar to Inter)

### Content Elements to Include
- 🧘 Buddhist symbols or imagery (respectful)
- 📜 Pecha/manuscript imagery
- 🤖 AI/technology elements
- ⚡ Modern, clean design
- 🌏 Professional appearance

### Platforms This Will Appear On
- WhatsApp link previews
- Facebook sharing
- Twitter cards
- LinkedIn posts
- Discord embeds
- Slack unfurls

### Quick Creation Options

#### Option 1: Use Existing Logo
If you have a larger version of `icon_logo.png`, you can:
1. Resize to 1200x630px
2. Add background and text overlay
3. Save as `og-image.png`

#### Option 2: Design Tools
- Canva (free templates available)
- Figma (web-based design)
- Adobe Express
- GIMP (free alternative)

#### Option 3: Online Generators
- OG Image Generator tools
- Social media template sites
- Buddhist-themed stock images + text

### Current Setup
The SEO system is already configured to use `/og-image.png` for:
- Open Graph `og:image`
- Twitter Card `twitter:image`
- Social media sharing previews

**Once you create and save the image as `public/og-image.png`, all social media sharing will automatically work with your branding!**
