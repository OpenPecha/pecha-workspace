# Open Graph Images for Pecha Tools

## Required Images

1. **og-image.png** (1200x630px)

   - Main Open Graph image for social media sharing
   - Should feature the Pecha Tools logo and tagline
   - Background: Buddhist manuscript or serene design
   - Text: "Pecha Tools - Buddhist Manuscript Digital Workspace"

2. **favicon.ico** (multiple sizes: 16x16, 32x32, 48x48)

   - Icon file for browser tabs
   - Should be based on icon_logo.png

3. **apple-touch-icon.png** (180x180px)
   - iOS home screen icon
   - Should be based on icon_logo.png

## Current Images

- `/icon_logo.png` - Main logo file (already exists)

## To Generate

You can create these images using:

1. Canva or Figma for og-image.png
2. Online favicon generators for favicon.ico
3. Image resizing tools for apple-touch-icon.png

**Dimensions**: 1200x630px
**Background**: Gradient from light blue to indigo with subtle Buddhist manuscript pattern
**Logo**: Pecha Tools logo (top left)
**Title**: "Buddhist Manuscript Digital Workspace" (large, centered)
**Subtitle**: "AI-Enhanced Tools for Scholars & Practitioners" (smaller, below title)
